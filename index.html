💭 Reflection & Learning Journey

📈 Growth from Lab 1 to Project 2

Starting from Lab 1, I built a strong foundation in Jetpack Compose. Lab 2 taught me state management, Lab 3 introduced Material Design, Lab 4 focused on navigation, and Lab 5 covered Room Database. Project 1 brought everything together into a multi-screen app.

For Project 2, I expanded my skills with advanced features: integrating Firebase Firestore for cloud storage, Retrofit for REST API calls (JobStreet via Apify), and GPS/Location sensors for "Jobs Near You". I also added chat functionality with Firestore sync and toggle save/unsave features.

⚡ Challenges Faced

  • Firebase Setup: Configuring `google-services.json` and enabling Firestore.
  • API Integration: Handling Retrofit calls and mapping data to UI models.
  • GPS Sensor: Permission handling and getting current location on emulator.
  • Chat Sync: Saving and retrieving messages from Firestore.
  • Profile Per Account: Fixing Room to store profile data by email.

🌍 SDG Impact

JobQuest supports SDG 1: No Poverty by providing a platform for job seekers to find employment opportunities. The app helps connect users with companies, streamlining the job application process and reducing barriers to employment. By helping individuals secure stable income, the app contributes to breaking the cycle of poverty.
